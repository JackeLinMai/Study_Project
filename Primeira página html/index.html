    • Estrutura básica do html

    Lembrando que as tags tem como função como delimitador de conteúdo. Sendo queas tags são identificas entre " <>"

    A estrutura básica de uma tem o seguinte formato:

        Html: Que serve para delimitar a págica e conteúdo.
        Head: O cabeçalho, o que se deve carregar antes, como o css ou algum script.
        Body: O que fica vísivel para o usuário, como por exemplo esse texto :)
        Footer: Que serve como rodapé de um site,levando a informações úteis, como contato,serviços e afins.

    Atributos das Tags

    Os atributos de uma tag servem para identificar o tipo, informar arquivos em que a tag tem que usar, o tipo de campo de texto, etc...

    Os tipos principais são:

        Id: Que é o identificar dela, muito útil quando se utiliza JavaScript.
        Src="caminho"
        type ="tipo", que é o tipo dela que poder ser Number, color, button,etc...

    Textos

    Os textos são escritos dentro de tags de "h1","h2","h3",etc.Sendo dado a importância em nível hierarquico sendo o h1 de maior relevância em um texto, como por exemplo no inicio da págica com o título "Introdução Prática com Html".

    Outras tags de textos são:

        p: comõe um paragráfo.
        b: conteúdo em negrito.
        span: legendas, pequenas informações.
        i: italico.
        br: quebra de linha.
        u: sublinhado.
        blackquote: citação
        sup: notinha de rodapé

    Como fazer listas?

    Uma lista odernada é delimitada pelas etiquetas "ol" e delimitada por "li".
    A estrutura da lista não ordenada é semelhante a da lista ordenada, entretanto, entre as tags "ul".

    Adicionando Links

    Links no html são conhecidos como Hiperlinks. Um link pode ser uma imagem de qualquer outro elemento.
    também é bom mencionar o atributo "a" que é o destino do link, e outro atributo seria "target" que expecifica onde abrir o documento vinculado.
